Your computer is given an IP address when it connects to the Terrace Network. (Private IP Address). When you open a browser and you connect to an URL. Your web browser makes a REQUEST from the computer with an IP Address that matches that URL.

The RESPONSE consists of packets that when assembled becomes html/images/video/sounds etc.

  • Local Link - a reference to an file location local to the current web page
  • Global Link - a reference to a resource across the Internet. It creates a request to another URL.
  • HTTPS - Packets are encrypted.
